## 🧨环境需求
- 需要mongodb数据库

## ✨部署
下载安装好数据库和一个Python3.10
1. 安装依赖
    ```cmd
   pip install -r requirements.txt
   ```
2. 运行启动脚本---run.cmd

这样你就可以试着发送一个**post**请求了，像这样**http://127.0.0.1/search?q=test**
一定要是**post**请求噢！！！
